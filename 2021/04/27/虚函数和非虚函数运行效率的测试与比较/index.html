虚函数和非虚函数运行效率的测试与比较
更新于 2021-04-27 C++

任务描述

英文版

Write a class with one virtual function and one non-virtual function. Inherit a new class, make an object of this class, and upcast to a pointer of the base-class type. Use the clock( ) function found in (you’ll need to look this up in your local C library guide) to measure the difference between a virtual call and non-virtual call. You’ll need to make multiple calls to each function inside your timing loop in order to see the difference.

粗略翻译

编写一个具有一个虚函数和一个非虚函数的类。继承新类，生成此类的对象，并向上转换为基类类型的指针。使用中的clock()函数（需要在本地C库指南中查找）来测量虚拟调用和非虚拟调用之间的差异。您需要对时间循环中的每个函数进行多次调用，才能查看差异。

思路分析

通过分别调用相同类的虚函数和非虚函数，记录并输出调用两种函数所需要的时间，通过比较输出时间的差异，来测试虚函数和非虚函数运行效率的差异。
